This assignment was a lot of fun! I enjoyed creating widgets with the ArcGIS JS API and using feature layers to fill the widgets, making the app more interactive.

On my original app, I struggled with querying both layers in the ArcGIS JS API and then filtering the cities using the continent layer. To work around it, I used an ArcGIS Python Notebook to map a dictionary and assign continents to cities, adding another perspective on data problem-solving using the ArcGIS ecosystem of tools. I did this to be able to mantain my focus on creating the dropdown and sidebar widgets while still keeping the overall functionalities of the feature layer, like pop-ups.

Even though I submitted the assignment quite late, it was a valuable experience. It taught me a bunch about web development.
